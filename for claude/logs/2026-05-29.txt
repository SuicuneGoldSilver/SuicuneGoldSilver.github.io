================================================================================
KEY LEARNINGS / NOTES
================================================================================

- @property CSS Houdini: allows animating custom properties (like --angle in
  conic-gradient). Supported in all modern browsers. Without it, you can't
  transition/animate inside gradient functions.

- Service worker scope: sw.js must be at the root of the site (same level as index.html)
  to have scope over all pages. GitHub Pages serves from root so this works correctly.

- fetchpriority on dynamic React content: must use MutationObserver since the browser
  has already started the fetch pipeline by the time React renders. Setting the attribute
  after the fact still influences the browser's priority queue for pending requests.

- Serebii image URL format: https://serebii.net/card/{setSlug}/{n}.jpg
  No zero-padding on the card number. Requires User-Agent + Referer headers.

- EN secret rare download approach: read numberEn from JSON to get EN card number,
  map setIdEn to Serebii slug, download and save as {jpNum}.jpg (not {enNum}.jpg)
  so the existing imageEn path in JSON stays correct.

- Dragon type in TCG is gold, not blue. Blue is Water territory.

- navigator.vibrate?.(10): optional chaining makes this safe everywhere.
  Only works on Android Chrome. iOS ignores it silently.

- PWA beforeinstallprompt: only fires on Android Chrome and desktop Chromium browsers.
  iOS Safari requires manual instructions. The event must be preventDefault()d to
  suppress the browser's own prompt before showing a custom one.
